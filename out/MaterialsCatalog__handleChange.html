Class: _handleChange

_handleChange(e, inputField, id)

new _handleChange(e, inputField, id)

Handles the changes to the selected material's properties.
Parameters:
Name Type Description
e event The event that is being sent through, use this to get the value in the input.
inputField string The property label for the input, use this to see which value is being changed.
id number The id of the currently selected materials item, use this to know which one to update in the list of materials array.
